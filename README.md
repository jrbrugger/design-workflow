# design-workflow

**Research-to-code automation for product designers.**

A collection of reusable Claude Skills and templates for structured design workflows: research planning, analysis, synthesis, and component generation. Built on "Just Enough Research" principles by Erika Hall.

Perfect for:
- Designers who want to own end-to-end workflows (research → design → code)
- Teams standardizing research and design processes
- Anyone tired of starting from scratch on every project

## What's Inside

### Skills (Claude Code)

**Available now:**
- **`/research-plan`** — Structure and write UX research briefs with 11 sections

**Coming soon:**
- `/interview-guide`, `/affinity-map`, `/persona-generator`, `/research-synthesis`

## How to Use
```
/research-plan

I'm planning research on [topic].
The problem is [problem].
Target users are [segment].
```

## Getting Started

1. Read `.claude/skills/research-plan/SKILL.md`
2. Copy `research-brief-template.md` into Notion
3. Run `/research-plan`
4. Review `research-brief-checklist.md`
