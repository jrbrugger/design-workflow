# Quick Start

Get a research brief written in 5 minutes.

## 1. Open Claude Code

Type: `/research-plan`

## 2. Describe Your Research
```
I'm planning research on [topic].
The problem: [problem statement].
Target users: [segment].
Main question: [research question].
```

## 3. Answer Clarifying Questions

Takes 2–3 minutes.

## 4. Get Your Brief

Copy → paste into Notion → done.

---

**Ready?** Type `/research-plan` 🚀
